# TGX

# TypeORM Toolkit Executable

Think of this as the prisma executable for typeorm, it helps;

1. handle migration generation
2. handle migration executing
3. handle seeding
4. handle creating db
5. handle dropping db

The package `typeorm-extension` already handles most of this but
the interface used isn't easy to use, easy to remember or having organized structure for
alot of its other functions.
